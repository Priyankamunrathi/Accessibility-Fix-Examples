# Accessibility Fix Examples

## 📌 Overview

This repository demonstrates common accessibility issues and their fixes using HTML and CSS based on WCAG 2.2 guidelines.

## 🔧 Topics Covered

* Missing alt text
* Keyboard accessibility
* Form labels
* Focus indicators
* ARIA attributes
* Heading structure

## 💡 Purpose

These examples help developers and testers understand how to fix accessibility issues in real projects.

## 🛠 Tools Used

* HTML
* CSS
* Chrome DevTools
